# Project Rahasaya

Project Rahasaya is a mysterious conspiracy investigation portal built using pure HTML and CSS.

The website is designed as a secret archive where users can explore paranormal cases,classified explainations, unusual incidents and also report them.

---

# 🚀 Phase 1 Progress

In Phase 1, the complete HTML structure of the website was created.

The focus was on:
- semantic HTML
- proper page structure
- forms
- tables
- lists
- accessibility
- content organization
